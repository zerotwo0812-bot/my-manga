1. Install all recommended plugins

Unless you are familiar with WordPress and our themes, it is recommended to install ALL required plugins including the suggested list which appears in admin panel and built-in plugins in the package (unzip the .zip package and find in /Installation Files/ folder)

When first installed, you will see this suggested list in admin. Click on the links to install and activate them. Other plugins can be found in the "Theme package\Installation Files\Plugins" as below
It is recommended to install all built-in plugins in the package. Go to Plugins > Add New > Uploads to install and activate them.

3. Start Configuring (Theme Options)

Appearance > Theme Options is where you do global settings including layout, font, color and other settings. Settings are organized in different tabs which affect different pages. For example, General tab contains site-wide settings while Archives tab contains settings for archives pages only (i.e. blog, category, tag, author, search pages)

Theme Options is where you do all the global configuration.

5. Manage Manga

To create a new Manga, please go to Manga > Add New . You can upload single chapter or multi chapter in this manga here

5.1. Upload Single Chapter Manga

There are 4 steps:

  • Step 1: Enter information such as Title, Summary and Select Featured Image.  
  • Step 2: Enter information of Manga Extra Info.  
  • Step 3: Go to > Chapter Upload Tab. After, You create New Volume >> Enter title Chapter Name and Name Extend ( Note: you can enter or don't enter title of Name Extend) >> Choose where to upload such as local,imgur, blogspot, amazone s3 ( If you select upload to imgur, blogspot, amazone s3, then you consult link below Manga storages >> Choose file and Upload Chapter
  • Step 4: Hit " Publish"
5.2. Upload Multi Chapter Manga

There are 4 steps:

  • Repeat Step 1 and Step 2 in Upload single Chapter Manga
  • Step 3: Go to > Manga Upload Tab. After, Choose Storage such as local,imgur, blogspot, amazone s3 ( If you select upload to imgur, blogspot, amazone s3, then you consult link below Manga storages >> Choose file and Upload Manga
  • Step 4: Hit " Publish"

6. Create More Complex Front Page

Time to move on! Make sure you have done Step 5. as we will continue from there. Edit Home page and start doing configuration

  1. Choose Front Page template for this page. (It is not required, but you will have more settings for the Front Page)
    Choose Front Page template to enable extra settings
  2. Front Page Settings box will appear under the content editor. Use these boxes to configure the Front Page.
  3. Use Widget to build your page sidebar content (go to > Appearance > Widgets.). Widget helps you use create better user interface. Here are the list of available Widgets.
  4. Now refer back to Step 4., you can configure position the sidebars by moving to Sidebar Setting Tab
  5. Now check back Front Page, does it look like our demo page? Try to change the settings, you will see (almost) unlimited number of combinations for your Front Page. Happy configuring!
